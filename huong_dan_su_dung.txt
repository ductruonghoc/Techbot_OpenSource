HƯỚNG DẪN SỬ DỤNG / DEPLOY VỚI DOCKER

1. **Yêu cầu hệ thống**
   - Đã cài đặt Docker trên máy. Nếu chưa, tham khảo hướng dẫn cài đặt tại file `huong_dan_cai_dat.txt`.

2. **Clone mã nguồn**
   - Clone dự án từ GitHub:
     ```bash
     git clone https://github.com/your-repo/techbot.git
     cd techbot
     ```

3. **Xây dựng Docker image**
   - Sử dụng lệnh sau để xây dựng Docker image:
     ```bash
     docker build -t techbot_image .
     ```

4. **Chạy container**
   - Sau khi xây dựng thành công, chạy container bằng lệnh:
     ```bash
     docker run -p 3000:3000 techbot_image
     ```

5. **Truy cập ứng dụng**
   - Mở trình duyệt và truy cập vào địa chỉ:
     ```
     http://localhost:
     ```

6. **Kiểm tra logs**
   - Nếu cần kiểm tra logs của container, sử dụng lệnh:
     ```bash
     docker logs <container_id>
     ```

7. **Dừng container**
   - Để dừng container đang chạy, sử dụng lệnh:
     ```bash
     docker stop <container_id>
     ```

8. **Xóa container**
   - Nếu cần xóa container, sử dụng lệnh:
     ```bash
     docker rm <container_id>
     ```

9. **Xóa Docker image**
   - Nếu cần xóa Docker image, sử dụng lệnh:
     ```bash
     docker rmi techbot_image
     ```

Nếu gặp lỗi trong quá trình sử dụng, hãy kiểm tra lại các bước hoặc liên hệ với nhóm phát triển
